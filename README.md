# Froxy-Megabonk

**Megabonk Cheat — Memory Hacking Tool**

Froxy-Megabonk is a simple memory hacking tool for the Megabonk game.  
It uses Cheat Engine‑style memory manipulation and exposes a basic Windows GUI, plus a raw Python version for learning. :contentReference[oaicite:1]{index=1}

---

## 🚀 Features

- Simple memory manipulation using Cheat Engine‑style offsets and pointers  
- Fixed pointer/offset logic  
- **C# Windows Forms GUI** (compiled and ready to use)  
- **Raw Python version** for learning and experimentation  
- Great for beginners who want to see how cheat memory access works :contentReference[oaicite:2]{index=2}

---

## 📦 Requirements

To compile or run from source:

- **Python 3.14**  
- `pymem` library  
- **SharpDevelop** or any C# compatible IDE (like Visual Studio) :contentReference[oaicite:3]{index=3}

To simply use the prebuilt tool:

- Download the latest `.exe` from **Releases** and run it.
